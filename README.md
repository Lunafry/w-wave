Одностраничный адаптивный веб-сайт для гипотетической радиостанции «W-Wave». Проект создан для портфолио в 2023г и демонстрирует современные подходы к фронтенд-разработке.

## Ключевые особенности

*   Полностью адаптивный дизайн: Оптимальное отображение на устройствах от 320px (мобильные) до 1920px+ (десктоп).
*   Семантическая верстка: Корректное использование HTML5 тегов (<'header>, <'section>, <'nav>, <'main>, <'footer> и др.) для улучшения доступности (accessibility) и SEO.
*   Доступность: Использованы ARIA-атрибуты (aria-label), осмысленные тексты для ссылок и кнопок, правильная структура заголовков.
*   Оптимизация изображений:
    *   Использование тега <'picture> с srcset для адаптивных изображений логотипа.
    *   Разные версии изображений гостей для мобильных и десктопных разрешений.
*   Чистая и модульная CSS-архитектура:
    *   Разделение стилей на style.css (основные), normalize.css (сброс) и media.css (медиазапросы).
    *   Использование методологии БЭМ для именования классов.
*   Интерактивные элементы: Стилизованные кнопки, кастомные чекбоксы, hover-эффекты для карточек и ссылок.
*   Валидная форма обратной связи с проверкой обязательных полей.

## Используемые технологии

*   HTML5 (семантическая разметка, валидация формы)
*   CSS3 (Flexbox, Grid, медиазапросы, кастомные свойства (CSS Variables), плавные переходы)
*   SVG (иконки и графические элементы)
*   Методология БЭМ (именование классов)
*   Normalize.css (для кроссбраузерной совместимости)
*   Git (система контроля версий)

## Адаптивность

Дизайн проработан для трех ключевых групп устройств:

*   Мобильные (Mobile): 320px — 767px
*   Планшеты (Tablet): 768px — 1023px
*   Десктоп (Desktop): 1024px и выше

Особое внимание уделено:
*   Гибкой сетке и переносу элементов.
*   Оптимизации навигационного меню для мобильных.
*   Адаптивным изображениям.

## Навигация по разделам сайта

1.  Главный экран (Hero): Введение и кнопка прослушивания эфира.
2.  Подкасты (Podcasts): Сетка последних выпусков подкастов.
3.  Передачи (Broadcasts): Список регулярных программ с расписанием.
4.  Гости (Guests): Карточки приглашенных экспертов.
5.  Плейлисты (Playlists): Коллекция тематических музыкальных подборок.
6.  Магазин (Wave): Презентация фирменной радио-колонки.
7.  О нас (About Us): Команда проекта и форма обратной связи.
